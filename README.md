# ReactTemplate
React template using API, Material UI.  

### Installing

```
 
clone https://github.com/BFronz/my-react-template.git

npm install package.json

Run  npm start

```

## Built With
* React.js
* React Hooks
* Material UI
* API data fetching 
* Node.js
