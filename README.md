# ReactTemplate
React template using API, Material UI. Work in progress. 

### Installing

```
 
clone https://github.com/BFronz/covidtracker.git

npm install package.json

Run  npm start

```

## Built With
* React.js
* React Hooks
* Material UI
* API data fetching 
* Node.js
